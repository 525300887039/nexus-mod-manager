## 新功能

### Nexus Mods 集成
- 浏览 Nexus Mods 热门/最新/最近更新的 STS2 Mod
- 查看 Mod 详情、文件列表、下载量和评分
- 内嵌浏览器直接下载并自动安装 Mod
- 本地 Mod 关联 Nexus 信息，提示可用更新

### 翻译系统升级
- 新增自定义大模型 API 翻译（兼容 OpenAI 格式）
- 三级翻译策略: 本地缓存 -> 免费 API -> 大模型
- SQLite 本地缓存，翻译过的内容不再重复请求
- 批量翻译 Nexus Mod 名称和描述
- 翻译设置面板: 引擎选择 + 大模型配置 + 缓存管理

### 其他改进
- 新增设置页面（Nexus API Key + 翻译配置 + 关于）
- 侧边栏新增 "Nexus 浏览" 和 "设置" 导航

## 升级说明
- 旧版翻译数据会自动迁移到 SQLite，无需手动操作
- 使用 Nexus 浏览功能需要配置 API Key（免费获取）
