## Name
Nexus Mod Manager

## Summary
A lightweight desktop manager for browsing, downloading, installing, and organizing Nexus Mods across multiple supported games.

---

## Description

[size=5][b]Nexus Mod Manager[/b][/size]

A modern desktop application for managing Nexus Mods workflows across multiple games. Browse popular mods, inspect details, download files, install archives, and keep different game setups separated inside one app.

[url=https://github.com/525300887039/nexus-mod-manager]GitHub Repository (Open Source)[/url]

[size=4][b]Features[/b][/size]

[list]
[*][b]Multi-Game Profiles[/b] - Keep game configuration, mod lists, and saved profiles isolated per game
[*][b]Nexus Browser[/b] - Browse popular, latest, and recently updated mods with in-app details
[*][b]Archive Install[/b] - Install ZIP, RAR, and 7Z archives with drag-and-drop support
[*][b]Enable / Disable[/b] - Toggle mods on and off without manually moving files
[*][b]Dependency Signals[/b] - Detect dependency issues and surface risk hints before launch
[*][b]Translation[/b] - Translate mod names and descriptions with local cache support
[*][b]Profiles[/b] - Save and restore different mod setups for the same game
[*][b]Per-Game Switching[/b] - Change the active game without rebuilding the app or rewriting settings
[/list]

[size=4][b]Installation[/b][/size]

[list=1]
[*]Download the latest build from the Files tab
[*]Run the installer or portable executable
[*]Launch "Nexus Mod Manager" and select the game you want to manage

[size=4][b]Requirements[/b][/size]

[list]
[*]Windows 10/11 (64-bit)
[*]A supported game installed locally
[*]WebView2 Runtime (included in installer, or pre-installed on Windows 11)
[/list]

[size=4][b]Privacy & Safety[/b][/size]

[list]
[*]Fully open source - [url=https://github.com/525300887039/nexus-mod-manager]view the code on GitHub[/url]
[*]No telemetry or analytics
[*]Network access is only used for Nexus APIs and optional translation providers
[/list]
